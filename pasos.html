Pasos para trabajr con git

Una vez clonamos la carpeta del compañero lo que tenemos que hacer es: a. Git pull para obtener los ultimos datos obtenidos. b. Creamos una nueva rama con git chekout -b "Nombre-de-la-nueva-rama". c. Hacemos el trabajo que nos toque. d. Tras hacer el trabajo tenemos que usar el comando Git add . para que se añada lo que hemos hecho. e. Añadimos un commit con el comando Git commit -m "Nombre del commit que queramos". f. Hacemos Git push para añadir el trabajo que hemos hecho al proyecto de GitHub. g. Si nos sale un error junto a un comando tendremos que copiar y pegar el comando que nos da la consola y en caso de que pida las credenciales introducirlas. h. Vamos a GitHub, tras esto, vamos a branches, una vez ahi tenemos que: -Ir a la nueva rama que se ha creado. -Crear una nueva pull request. -Pasar el enlace al compañero para que pueda ver los cambios realizados. -Estos, tendrán que ir a Files changed donde deberan, dejar un comentario y aprobar los cambios dando click en "Viewed". i. Cuando los compañeros aprueben los cambios se puede hacer el Merge.
